Html Tables, Exercise 1

Add a table row with two table headers.

The two table headers should have the value "Name" and "Age".

    
        <table> 
            <tr> 
                <th>Name</th> 
                <th>Age</th> 
            </tr> 
            <tr> 
                <td>Jill Smith</td> 
                <td>50</td> 
            </tr> 
        </table>

Example:

Name Age
Jill Smith 50
