Html classes , Exercise 2

Add the correct class to make the H1 element red.

"special" class.

    
        <!DOCTYPE html> 
        <html> 
            <head> 
                <style> 
                    .mystyle {color:red;} 
                </style> 
            </head> 
            <body> 
                <h1 class="mystyle">My Home Page</h1> 
            </body> 
        </html>
