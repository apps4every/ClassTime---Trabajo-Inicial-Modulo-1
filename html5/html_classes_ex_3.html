Html classes , Exercise 3

Add the correct class to make the H1 element red.

"special" class.

    
        <!DOCTYPE html> 
        <html> 
            <head> 
                <style> 
                    .intro {background:pink;} 
                    .special {color:red;} 
                </style> 
            </head> 
            <body> 
                <h1 class="intro special">My Home Page</h1> 
            </body> 
        </html>
