CSS font, Exercise 1

Set the font family for the page to "Courier New", and the font family for <h1> to "Verdana".


<html>
<head>
<style>
body {
  font-family: "Courier New";
}

h1 {
  font-family: Verdana;
}
</style>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

</body>
</html>
