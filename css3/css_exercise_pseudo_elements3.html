CSS pseudo_elements, Exercise 3

Insert the image "smiley.gif" before, and after <p> elements, using the 


<html>
<head>
<style>
p::before{
  content: url(smiley.gif);
}
p::after{
  content: url(smiley.gif);
}
</style>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

</body>
</html>
